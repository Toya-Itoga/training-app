# IRONLOG — 画面ハンドオフドキュメント

筋トレ記録 WEB アプリのフロントエンド画面仕様（モック）。  
本ドキュメントは Claude Code 等での実装着手時の参照用。

## 全体構成

| 領域 | 内容 |
| --- | --- |
| サイドバー (220px固定) | ナビゲーション (ダッシュボード / 記録 / 履歴 / 種目管理) + ユーザー情報 + ログアウト |
| メイン領域 | 各画面コンポーネント。**ページスクロール禁止**・コンテンツは `100vh` 内に収める設計 |

## 共通仕様

- フォント: `Anton` (display), `Space Grotesk` (UI), `JetBrains Mono` (数値・ラベル)
- 背景: `#060606` / `#0c0c0c` ベース、アクセントは Tweaks で可変（既定: 赤 `#ff3b30`）
- 単位: 重量は `kg` 既定、ユーザー設定で `lb` 切替（種目マスタ毎）
- 認証: JWT、ローカル保持 24h（モックではフロント state のみ）

## 画面別仕様

### 1. ログイン画面 (`screens/Login.jsx`)
- 左ペイン: ブランド + キャッチコピー、右ペイン: フォーム
- 認証成功で `onLogin(username)` → ダッシュボードへ遷移
- TODO: 実 API、パスワードリセット、新規登録、2FA

### 2. ダッシュボード (`screens/Dashboard.jsx`)
- 今週の統計サマリ × 3（回数 / 総ボリューム / 連続日数）
- 週間ボリュームバーチャート（月〜日、当日ハイライト）
- 直近 3 セッションリスト（クリックで履歴画面へ）
- CTA: 「新規記録」「続きを記録」 → 記録画面
- データ集計元: `HISTORY` グローバル fixture

### 3. トレーニング記録 (`screens/Record.jsx`)
- メタ: 日付選択のみ（記録項目は日付のみ、セッション名なし）
- 種目追加モーダル: 部位フィルタ、検索、過去ベスト表示
- 種目ブロックごとに: セット動的追加/削除、レップ・重量入力、メモ
- サイドパネル (240px): 総ボリューム / セット数 / 種目数 / 先週同日比 / 前回ベスト
- **プロップス**: `initialDate` で履歴画面からの日付引継ぎに対応
- 保存ボタンで完了トースト表示
- TODO: POST /api/workouts、種目 D&D 並び替え、前回からのコピー、タイマー

### 4. 履歴 (`screens/History.jsx`)
- 月別カレンダー（月曜開始、記録のある日に部位タグ表示）
- 日付クリック → 右パネルに詳細表示
- 詳細パネル: 集計 (ボリューム/セット/種目数) + 種目別セット内訳 + メモ
- 「編集」ボタンで該当日付を引き継いで記録画面へ遷移（`onEditDate(iso)` → `setRecordDate`）
- TODO: GET /api/workouts、週/リスト切替、フィルタ、CSV エクスポート

### 5. 種目管理 (`screens/Exercises.jsx`)
- 部位別グルーピング（胸/背中/脚/肩/腕、**種目0でも常に5部位表示**）
- 検索（日本語/英名）、部位フィルタチップ
- 各種目: 編集・削除モーダル
- 空部位ヘッダから「最初の種目を追加」CTA
- TODO: CRUD /api/exercises、サムネイル、部位追加、削除時の整合性チェック

## ルーティング・状態管理

| 状態 | 場所 | 役割 |
| --- | --- | --- |
| `loggedIn` | `App` | 認証ガード |
| `user` | `App` | ユーザー表示名 |
| `route` | `App` | 現在画面 ID |
| `recordDate` | `App` | 履歴 → 記録の日付引継ぎ用 |

API 連携時は React Query / SWR、認証は HTTP-only Cookie + JWT を推奨。

## デザイントークン (`styles.css` `:root`)

| 変数 | 用途 |
| --- | --- |
| `--accent` | アクセント色（Tweaks で可変） |
| `--bg-0` 〜 `--bg-4` | サーフェスランプ（暗→明） |
| `--fg-0` 〜 `--fg-4` | テキストランプ |
| `--line` / `--line-strong` | ヘアライン / 強調ボーダー |
| `--font-display` / `--font-ui` / `--font-mono` | フォント |
| `--r-sm` / `--r-md` / `--r-lg` | 角丸 |

## バリアント

Tweaks パネル右下から切替可能:
- **EDITORIAL** (既定): Anton 大見出し、柔らかい角丸
- **TERMINAL**: 全タイプ Mono、角丸ゼロ、ターミナル風

## アクセシビリティ・残課題

- フォーカスリング: 既定スタイルに依存（要カスタマイズ）
- キーボードナビゲーション未検証
- スクリーンリーダー向け aria-* 属性未付与
- レスポンシブはデスクトップ専用（モバイル別ビュー要設計）
